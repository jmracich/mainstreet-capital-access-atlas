Open-source collaboration

Contribute

Help improve data quality, local context, accessibility, documentation, and state spotlights. The most useful contributions make the atlas more careful and easier to verify.

Report a data issue

Share the county FIPS, affected field, observed value, expected value, and public source link.

Add local resource data

Contribute CDFI, chamber, SBDC, SCORE, library, community college, or local partner resource lists with clear provenance.

Suggest methodology improvements

Propose better indicators, source handling, missing-data treatment, or state-specific context with supporting evidence.

Add a state spotlight

Use the Illinois page as a model for a careful, locally relevant state view that explains why counties are featured.

Improve county brief language

Help keep generated text humble, non-stigmatizing, and clear about what public data can and cannot show.

Run the project locally

Install the package, run the public-data pipeline, execute tests, and preview the static site before opening a contribution.

Contribution principle

This project uses public data to start better conversations. It does not replace local knowledge. Contributions should make that principle clearer, not louder.
