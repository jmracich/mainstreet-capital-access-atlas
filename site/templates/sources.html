Reuse responsibly

Attribution and citation

Cite this open-source project separately from the underlying public datasets. The source manifest lists dataset names, publishers, URLs, access methods, limitations, and current build status.

Project citation

Use the repository citation file when available, and include the generated data package date when publishing analysis from county exports.

Source attribution

Preserve attribution for Census, FDIC, and any manual public sources added to the build. Source publishers do not endorse project interpretations.

Map attribution

The interactive map uses Leaflet and OpenStreetMap tiles. Keep map attribution visible when reusing screenshots or map views.
