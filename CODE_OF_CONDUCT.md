# Code of Conduct

This project follows a simple standard: contributors should treat each other and the communities represented in public data with respect.

## Expected Behavior

- Use welcoming, professional language.
- Assume data has limits and local context matters.
- Discuss disagreements with evidence and care.
- Avoid stigmatizing labels for people, places, and communities.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Doxing or sharing private information.
- Using the project to target, shame, or stereotype communities.
- Claims that the project determines creditworthiness, lending eligibility, or community value.

Maintainers may remove comments, issues, pull requests, or contributors that undermine these standards.
