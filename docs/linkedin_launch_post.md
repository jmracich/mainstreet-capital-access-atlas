# LinkedIn Launch Post Draft

I built Main Street Capital Access Atlas, an open-source public-data project designed to help local partners better understand where small-business opportunity, capital-access signals, community stress, and support resources may overlap.

The idea is simple: public data is useful, but it is scattered. County Business Patterns, FDIC branch data, Census context, CRA/SBA/CDFI signals, and housing indicators each tell only part of the story. This project brings those signals into cautious county-level "Main Street Opportunity Briefs" that can help community partners ask better questions.

The project is intentionally humble. It is not a credit model, not an underwriting tool, and not a substitute for local knowledge. It is a conversation starter for local verification, listening, resource mapping, and partnership.

The first spotlight is Illinois, with a national county-level architecture designed for expansion. The site is fully static, free to host on GitHub Pages, and built with Python, public data, and transparent methodology.

I am sharing it as an open-source civic-tech project because better public-interest tooling should be reusable. Contributions are welcome, especially around data quality, local resource mapping, accessibility, documentation, and new state spotlights.
